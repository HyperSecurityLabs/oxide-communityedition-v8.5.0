# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 7.7.x | ✅ Supported |
| < 7.7 | ❌ Not supported |

## Reporting a Vulnerability

If you discover a security vulnerability in OXIDE, please report it responsibly:

1. **Do not** disclose the issue publicly until it has been addressed.
2. Open a private report via [GitHub Issues](https://github.com/HyperSecurityLabs/OxideCE-v7.7.7ELITE/issues) with the label `security`.
3. Alternatively, reach out via [Telegram](https://t.me/hypersecurity_offsec).

We aim to acknowledge receipt within 48 hours and provide a fix timeline within 7 days.

## Responsible Disclosure

We follow a 90-day disclosure window. Researchers who report valid issues will be credited in release notes.
